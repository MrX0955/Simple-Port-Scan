# Simple-Port-Scan
Simple Port Scanner 

Usage = Port Scan.py [-h] [--ip IP] [--port PORT]

Don't work https:// or http:// searchs.                                                                                                                                                                         
Don't work port 0 or GreaterThan 65536. Just between 1-65535.
